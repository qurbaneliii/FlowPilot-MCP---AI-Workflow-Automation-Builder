# LinkedIn Post Draft

I built FlowPilot MCP as a lightweight AI workflow automation engine for a GitHub Repository Audit MVP.

Instead of treating the interaction like a chatbot, FlowPilot turns a natural-language request into a validated workflow graph, executes typed nodes, pauses before risky actions, and returns auditable artifacts.

The demo path reads a repository snapshot, analyzes documentation and project readiness, drafts guarded GitHub issues, requires human approval before issue creation, and generates markdown reports plus a LinkedIn-ready draft.

Default local mode is intentionally safe: fake agents, mock GitHub MCP, explicit mode labels, and no real external writes unless configured.
